## Setup

```
git clone https://github.com/MobinaJafarian/Laravel_Restaurant_Reservation.git 
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
```
Then

```
php artisan serve
```
```
npm install
npm run dev
```
